# LexyVoz - Proyecto React Native

# Empezar 
Prerrequisitos
Node.js LTS y pnpm/npm
Expo CLI (npm i -g expo)
(Opcional) EAS CLI (npm i -g eas-cli) para builds
Android Studio / Xcode si vas a emular

# Instalación
  # 1) Clonar
  git clone https://github.com/Shorkka/Lexyvoz.git
  cd Lexyvoz
  
  # 2) Instalar dependencias
  npm install

# Variables de entorno
  # 1.- Copia el template y editalo: 
        cp .env.template .env
  # 2.- Completa los varolres necesarios.
   El archivo .env.template existe en el repo como referencia.
# Corre en desarrollo 
  npm run start
